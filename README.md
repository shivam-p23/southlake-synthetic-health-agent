# Southlake Synthetic Health Agent

Hackathon project for the Southlake Regional Health Centre Innovation Challenge.

This project builds an **agentic synthetic healthcare data service** that:

• audits and cleans healthcare datasets  
• generates a synthetic patient population  
• assigns predictive risk scores  
• routes patients through a Distributed Health Network  
• simulates policy scenarios that reduce hospital pressure  

The system allows healthcare planners to explore strategies without using real patient data.

## Pipeline

Public Health Data  
↓  
Data Hygiene + Governance Check  
↓  
Synthetic Population Generator  
↓  
Risk Profiling Engine  
↓  
Care Navigation Engine  
↓  
Digital Twin Policy Simulator
